Video processing for the web side

Background

A H.264 video stream is composed of NAL units (NALU). Each NAL unit starts with [0x00, 0x00, 0x00, 0x01]1. Our only objective is to split these unit using this delimiter to get the individual NAL units, which the web server can then send to its clients.

  1. https://yumichan.net/video-processing/video-compression/introduction-to-h264-nal-unit/ ↩
