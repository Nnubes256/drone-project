pub struct RawAxisState {
    roll: f32,
    pitch: f32,
    yaw: f32,
    throttle: f32,
}

Raw processed axis state.

Serves as the final output of the gamepad logic, but it is formatted as floating-point numbers on the range [-1, 1] (except for the throttle, which works on the range [0, 1]).
