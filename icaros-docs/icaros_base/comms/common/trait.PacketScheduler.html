A packet scheduler.

Implementors of this trait can act as packet schedulers, middleware between device drivers and the core systems whose purpose is to schedule the input/output of packets.

I represents the type of the incoming packet (i.e. packet that updates state) O represents the type of the outgoing packet (i.e. packet that replicates state) S represents the type of the state being written or read to/from.
