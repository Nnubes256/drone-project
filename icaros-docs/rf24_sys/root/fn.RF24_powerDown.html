pub unsafe extern "C" fn RF24_powerDown(this: *mut RF24)

Enter low-power mode

To return to normal power mode, call powerUp().

@note After calling startListening(), a basic radio will consume about 13.5mA at max PA level. During active transmission, the radio will consume about 11.5mA, but this will be reduced to 26uA (.026mA) between sending. In full powerDown mode, the radio will consume approximately 900nA (.0009mA)

@code radio.powerDown(); avr_enter_sleep_mode(); // Custom function to sleep the device radio.powerUp(); @endcode
